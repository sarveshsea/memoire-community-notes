# Apple Desktop Automation

Prepare macOS agents to work with Apple apps, local files, permissions, and desktop automation.

## When to Use

Use this Note when a Memoire or Studio run needs macOS automation, Apple Notes, Reminders, iMessage, desktop app control.

## Workflow

1. Check Accessibility, Screen Recording, Automation, and file permissions before driving desktop apps.
2. Prefer app-specific CLIs or Shortcuts before using broad GUI automation.
3. Keep personal app access opt-in and scoped to the named task.
4. Capture screenshots or structured app output after important desktop actions.
5. Never run destructive desktop actions without a visible confirmation step.
