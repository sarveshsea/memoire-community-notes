# Sync with Notion

Read [the Notion integration guide](references/guide.md). Confirm the connected workspace, target database, properties, source of truth, and write scope before mutation. Map fields explicitly, validate external data, preserve stable IDs, and make retries idempotent. Never expose integration secrets. End with counts, skipped records, conflicts, and links or identifiers for the updated records.
