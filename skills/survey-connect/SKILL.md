# Connect survey data

Read [the survey integration guide](references/guide.md). Confirm consent, data sensitivity, source schema, response identifiers, and desired outputs before import. Preserve raw data, normalize into a separate derived dataset, validate row counts and missing values, and redact personal data when it is not required. Report transformations, exclusions, and data-quality limitations before synthesis.
