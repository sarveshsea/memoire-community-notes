# Web research

Read [the structured research guide](references/guide.md). Convert the request into answerable questions, define source quality and freshness requirements, and search primary sources first. Record claim-level evidence, distinguish publication date from event date, and label inference. Stop when additional sources no longer change the conclusion. Return the answer, material uncertainty, and direct source links.
