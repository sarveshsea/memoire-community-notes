# Web research

Read [the structured research guide](references/guide.md). Convert the request into answerable questions, define source quality and freshness requirements, and search primary sources first. Record source URL, access date, claim, confidence, and time sensitivity in an evidence table. Use screenshots for visual claims and direct API responses for machine-readable facts. Distinguish publication date from event date, separate quotations from paraphrases, and label inference. Stop when additional sources no longer change the conclusion. Return the answer, material uncertainty, unresolved verification gaps, and direct source links.
